CoProc2 Service Installation

The CoProc2 service must be installed on an accessible DataPower appliance before any sytlesheets can be executed by CoProc2. Fortunately, this is a simple as

  1. Obtaining the CoProc2 export.
  2. Importing it into a DataPower appliance's application domain.

Note the port on which the CoProc2's FSH is listening. This will be needed for the run configuration.
